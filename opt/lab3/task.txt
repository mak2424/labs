Для выполнения лабораторной работы 3 Вам понадобится ознакомиться с библиотекой GSL (GNU Scientific library)
https://www.gnu.org/software/gsl/doc/html/intro.html и BLAS (Basic Linear Algebra Subprograms) http://www.netlib.org/blas/.
Библиотека BLAS представляет набор готовых функций для векторных и матричных операций, оптимизированных для работы на большинстве процессорных архитектур.
Библиотека GSL предоставляет более удобный доступ к функциям BLAS библиотеки.

Перепишите код из лабораторной работы 2, заменив в нем функцию matmult_native на аналог из библиотеки CBLAS.
Замерьте скорость выполнения, сравните с результатом из предыдущей работы.

Для выполнения работы при каждом входе на кластер следует подключать актуальную версию компилятора, для этого выполните команду:
module load gcc
Проверить текущую версию можно командой:
gcc -v
Актуальная версия, установленная на кластере 5.3.0
